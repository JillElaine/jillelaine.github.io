idleTimeout Testing Page

The idle time limit, 'idleTimeLimit', is set to 45 seconds (45000 milliseconds) for this demonstration.

Activity events to detect, 'activityEvents', are set to 'click keypress scroll wheel mousewheel'. The event, 'mousemove', has been removed for this demonstration.

If no activity events are detected within the idle time limit of 45 seconds, the warning dialog will appear.

The warning dialog display limit, 'dialogDisplayLimit', is set to 3 minutes (180000 milliseconds). If no activity is detected within the 'dialogDisplayLimit', the page will redirect.
